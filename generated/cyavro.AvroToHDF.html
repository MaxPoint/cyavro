cyavro.AvroToHDF¶

class cyavro.AvroToHDF¶

Utility namespace class that provides methods to convert Avro to HDF5 for easier use within the rest of the python ecosystem.

This implementation will chunk the avro reads and write these chunks using multiple threads. This will not read the entire avro file into memory at once. This allows for safe use when converting large avro files to HDF5.

Structured as a class so that consumers can subclass this if they wish to override behavior
