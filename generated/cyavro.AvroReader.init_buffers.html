cyavro.AvroReader.init_buffers¶

AvroReader.init_buffers(self, size_t chunk_size=0)¶

Initialize the buffers for the reader object. This must be called before calling read_chunk()

This method also initializes and parses the avro schema for the file in question.

Parameters:

chunk_size : size_t

Number of records to read in chunk. This essentially determines how much memory we are allowing the reader to use.
