cyavro.AvroReader¶

class cyavro.AvroReader¶

Avro Reader class for reading chunks from an avro file.

Parameters:

filename : str

Input filename to be read

Examples

>>> rd = AvroReader('/path/to/file.avro')
>>> rd.init_reader()
>>> rd.init_buffers(10000)
>>> while True:
...     df = rd.read_chunk()
...     ...
...     # Do something with df
...     ...
...     if len(df) == 0:
...         break
...
>>> rd.close()
